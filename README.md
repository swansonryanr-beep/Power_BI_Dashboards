# My Power BI Dashboard Portfolio

Hello, this repository is a collection of Power BI dashboards that I have developed. It documents my progression in learning Power BI, from foundational reporting concepts to more advanced dashboard design and analytics techniques.

The goal of this portfolio is not only to showcase technical skills, but also to demonstrate how data can be transformed into useful, clear, and actionable insights.
